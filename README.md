# orders-api

### About
This is a simple Web Service for orders.
Registered users can
- Place orders.
- View their own orders
- View all orders (Admin)
- Update / Confirm an order.




### Available endpoints.
- GET /orders/mine       
- GET /orders/all
- PATCH /orders/confirmation
- POST /orders/new

- POST /users/login
- POST /users/register
